Задача размещения предприятий и выбора цен на их продукцию.

Формат файла
 - Первая строка: количество предприятий(m), количество клиентов(n), количество открытых предприятий.
 - Следующие m строк: матрица m*n затрат i-го клиента на доставку с j-го предприятия.
 - Последняя строка: вектор бюджетов клиентов.
